# Sales Data Analysis Dashboard (Excel & Power BI)

## Project Overview
This project analyzes historical pizza sales data to identify key revenue drivers, customer purchasing behavior, seasonal trends, and product performance. The analysis was carried out using Microsoft Excel and Power BI to convert raw sales data into meaningful insights through interactive dashboards.

The project demonstrates an end-to-end data analytics workflow including data cleaning, exploratory data analysis, KPI tracking, visualization, and business recommendations.

## Problem Statement
The objective of this project was to analyze a company’s pizza sales data in order to:
- Identify top- and bottom-performing pizzas
- Track overall sales performance using KPIs
- Understand monthly and category-wise sales trends
- Provide actionable business insights to improve revenue and operations

## Tools & Technologies
- Microsoft Excel (Data Cleaning, PivotTables, PivotCharts, Slicers)
- Power BI (Data Modeling, DAX Measures, Interactive Dashboards)
- Power Query (Data Transformation)

## Key KPIs
- Total Revenue: ₹817.86K
- Total Orders: 21,350
- Total Pizzas Sold: 49,574
- Average Order Value: ₹38.31
- Average Pizzas per Order: 2.32

## Dashboard Features
- Sales analysis by pizza category, size, and month
- Top 5 and Bottom 5 pizzas by revenue, quantity, and orders
- Monthly revenue trend analysis
- Busiest days and time analysis
- Interactive slicers for dynamic filtering
- Clear KPI cards and visual storytelling

## Key Insights
- The Classic category generated the highest revenue, followed by Supreme
- The Veggie category showed comparatively lower revenue
- Sales peaked in August and declined in September, indicating seasonal trends
- Classic Deluxe and Barbeque Chicken pizzas consistently performed well
- Certain pizzas showed low demand and may require promotional strategies

## Business Recommendations
- Focus marketing efforts on top-performing pizzas
- Introduce combo offers during peak demand periods
- Optimize inventory planning during high-sales months
- Review pricing or replace consistently low-performing items
